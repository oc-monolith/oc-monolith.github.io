monolith

monolithic kernel for opencomputers

github | documentation | installer

monolith is a monolithic kernel turned full-fledged operating system written in lua for the opencomputers minecraft mod.  it supports cooperative mulittasking through use of lua's coroutines.

installation

the monolith installer will run on both openos and monolith.  you will need a hard disk with either operating system installed--note that both disks must be writable or installation will fail.

in the operating system's shell, type wget https://raw.githubusercontent.com/ocawesome101/oc-monolith/master/installer.lua and press the [enter] key.  once the file has downloaded, run ./installer.lua and follow the prompts.  note that the installer will not let you install monolith to the tmpfs or the boot address, usually the currently mounted root filesystem.
